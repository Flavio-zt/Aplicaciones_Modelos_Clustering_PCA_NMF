# Aplicaciones_Modelos_Clustering_PCA_NMF
Descripción

Este repositorio contiene el desarrollo de la Actividad 05, cuyo objetivo es comprender las aplicaciones de diferentes técnicas de Machine Learning mediante el uso de datasets proporcionados por scikit-learn.

Durante la actividad se implementaron tres casos de estudio enfocados en el agrupamiento de datos, la reducción de dimensionalidad y los sistemas de recomendación, permitiendo analizar el funcionamiento y la utilidad de cada técnica en problemas reales.

Contenido del repositorio
Clustering_Jerarquico_20Newsgroups.ipynb
Aplicación de Clustering Jerárquico sobre el dataset 20 Newsgroups.
Comparación de diferentes criterios de linkage.
Análisis de los resultados obtenidos.
PCA_Olivetti_Faces_Eigenfaces.ipynb
Implementación de un sistema de reconocimiento de rostros utilizando el dataset Olivetti Faces.
Aplicación del método PCA (Análisis de Componentes Principales) mediante la técnica de Eigenfaces.
NMF_Sistema_Recomendacion.ipynb
Desarrollo de un sistema de recomendación de artículos.
Aplicación de NMF (Non-negative Matrix Factorization) y similitud del coseno para encontrar documentos relacionados.

Objetivos
Aplicar técnicas de Clustering Jerárquico.
Comprender el uso de PCA para la reducción de dimensionalidad.
Implementar NMF para la recomendación de artículos similares.
Analizar e interpretar los resultados obtenidos en cada caso de estudio.
